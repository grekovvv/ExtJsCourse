Подключение к проекту

1. Либо скачать отсюда локально папку с фреймворком https://www.sencha.com/products/extjs/communityedition/

Community Edition рапротраняется бесплатно по лиценции GPL v3, то есть ваши проекты должны выкладыватсья с ссылкой на исходный код программы и каждый может её скачать и изменять. Также эта версия предполагает, что вы можете заработать с помощью программ, написанных на ней до 10 000$. Если больше, нужно менять лицензию на платную

Компоновка

1. В ExtJS 4 компоненты не могу находиться сами по себе, они обязательно должны заключаться в какой-нибудь контейнер. Контейнер - это по сути тоже компонент, который может включать в себя дочерние компоненты и управлять их расположением. Наиболее распространенные контейнеры - Panel, Viewport, Window.
2. Компоненты внутри растягиваются относительно значения свойства layout. По умолчанию оно 'fit'. То есть, компоненты растягиваются по по всему размеру панели;
3. layout: vbox - располагает элементы по вертикали.
4. layout: hbox - располагает элементы по горизонтали.
5. Можно управлять находдением каждого элемента относительно границ - это регулирует параметр 'pack' со значениями start, center, end. Относительно заданого параметра все элементы прижмутся к заданной границе.
6. Таблицы. 7. Аккордеоны. 8. Слайды. 9. Анкорная (процентное соотношение размеров относительно родителя). 10. Граничная (аля ваша IDE)
