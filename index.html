Компоновка

  1. В Ext JS 4 компоненты не могут существовать сами по себе, они обязательно должны быть заключены в какой-то контейнер. Контейнер – это, по сути, тоже компонент, который может содержать дочерние компоненты и управлять их расположением. Наиболее распространённые контейнеры – Panel, Viewport, Window.
  2. Компоненты внутри контейнера растягиваются относительно значения свойства layout. По умолчанию оно равно 'fit', то есть компоненты растягиваются на весь размер панели.
  3. layout: vbox – располагает элементы по вертикали.
  4. layout: hbox – располагает элементы по горизонтали.
  5. Можно управлять положением каждого элемента относительно границ с помощью параметра 'pack' со значениями start, center, end. Все элементы при этом прижимаются к указанной границе.
  6. Таблицы.
  7. Аккордеоны.
  8. Слайды.
  9. Анкорная (процентное соотношение размеров относительно родительского элемента).
  10. Граничная (аналогично вашей IDE).

Компоненты

  1. Компоненты сами по себе не могут существовать. Все они помещаются в некоторый контейнер. Контейнер представляет специальный компонент, который может включать другие компоненты. Стандартная, но не обязательная, иерархия компонентов может начинаться с класса Viewport, который затем включает остальные контейнеры и компоненты.
  2. Каждый контейнер имеет свойство items, в котором можно установить дочерние элементы.
  3. Рендеринг компонентов. renderTo. Направленный и отложенный (direct, lazy).
  4. Добавление, отображение и удаление элементов.
  5. Виды компонентов.
